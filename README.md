💻 Sobre o projeto

MEULINK é um aplicação para o encurtamento de links.

⚙️ Funcionalidades

*Adicionar link
*Encurtar o link
*Listar links ja encurtados
*Apagar links da lista de links

🛠 As seguintes tecnologias foram utilizadas no desenvolvimento da API Rest do projeto:

*JavaScript
*Create React App
*React.js
*Axios

📄 Libraries React:

*React router dom
*React icons
